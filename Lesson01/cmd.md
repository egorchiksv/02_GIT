# Подсказки по командной строке

Команда смены деректории 
```sh
cd c:\folder_name
```

Команда отображения текущей деректории. MacOs, Linux
```sh
pwd
```

Листинг текущей деректории
Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
```
Удаление файла в Windows:
```sh
del <filename>
```
в Linux, MacOs:
```sh
rm <filename>
```
